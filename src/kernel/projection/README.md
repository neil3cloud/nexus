# Kernel Projection

## Purpose

Reserve the kernel boundary that materializes deterministic mission-scoped projections from evidence.

## Responsibilities

- Assemble mission context from authoritative evidence
- Compute deterministic projection views
- Publish context packages for execution and review

## Scope

Focused on projection contracts and lifecycle semantics, not projection cache implementation.

## Relationship to Shared Reality

Projection operationalizes Shared Reality as a deterministic computed view.
