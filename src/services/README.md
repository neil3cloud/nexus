# Services

## Purpose
Reserve application services that compose lower-level contracts without becoming hosts.

## Responsibilities
- Coordinate reusable flows
- Encapsulate cross-module operations
- Remain aligned with kernel boundaries

## Scope
Services should remain thin and policy-aware.

## Future Evolution
Future work may introduce focused services for corpus enrichment, planning support, or evidence retrieval.

## Relationship to the Engineering Corpus
Services can help operationalize Shared Reality while preserving capability boundaries.
