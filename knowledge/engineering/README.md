# Engineering Practice Corpus

## Purpose
Capture how Nexus expects engineering work to be performed in a Shared Reality Platform.

## Responsibilities
- Define evidence-oriented engineering practices
- Explain how work should enrich the corpus
- Document lifecycle expectations for humans and AI systems

## Scope
Focused on engineering method rather than feature implementation.

## Future Evolution
These documents should become more concrete as operating practices emerge across hosts and capabilities.

## Relationship to the Engineering Corpus
Engineering practice documents explain how Shared Reality is assembled, trusted, and improved through day-to-day work.
