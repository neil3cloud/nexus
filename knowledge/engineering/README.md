# Engineering Practices

## Purpose

Capture how Nexus expects engineering work to be performed in a role-based engineering workspace.

## Responsibilities

- Define evidence-oriented engineering practices
- Explain how work should improve future engineering sessions
- Document lifecycle expectations for humans and AI systems

## Scope

Focused on engineering method rather than feature implementation.

## Future Evolution

These documents should become more concrete as operating practices emerge across hosts, roles, and workflow slices.

## Relationship to Shared Reality

Engineering practice documents explain how Shared Reality is assembled, trusted, and improved through day-to-day work.
