# Interface Contracts

## Purpose

Define architecture-level contracts between the Host boundary, Kernel services, Event Bus, and Adapter boundary.

## Scope

These contracts define structural interfaces and message shapes at a reference-architecture level.

They do not prescribe implementation technology, protocol bindings, or runtime libraries.
