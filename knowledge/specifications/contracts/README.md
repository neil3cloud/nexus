# Shared Contracts

## Purpose

Collect shared contracts used across kernel, adapters, hosts, and providers.

## Responsibilities

- Describe cross-layer interfaces
- Clarify boundary expectations
- Reduce accidental coupling

## Scope

Focused on workspace-facing contracts rather than implementation details.

## Future Evolution

Future work may split contracts by stability level, ownership, and versioning policy.

## Relationship to Shared Reality

Explicit contracts help preserve Shared Reality consistency across workspace layers.
