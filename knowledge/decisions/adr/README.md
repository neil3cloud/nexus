# Architecture Decision Records

## Purpose

Store formal architectural decisions that shape Nexus over time.

## Responsibilities

- Capture decision context, alternatives, and consequences
- Preserve rationale for future reference
- Provide a stable citation target for later design work

## Scope

Contains durable architectural decisions, not implementation notes.

## Future Evolution

Future ADRs should reference foundational decisions when they extend or constrain workspace direction.

## Relationship to Shared Reality

ADRs are part of the evidence used to assemble Shared Reality because they make important reasoning explicit and reusable.
